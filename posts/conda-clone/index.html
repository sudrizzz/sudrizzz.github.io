Anaconda 克隆到离线环境

Aug 05, 2021
1 minute read

配置 conda 环境

首先我们需要在在线环境中配置好 conda 环境，安装好所需要的包，方可开始打包。此处假设我们创建的环境名为 baseenv。

打包

首先进入 Anaconda 的安装目录，将以下目录进行打包：

  1. anaconda3\envs\baseenv
  2. anaconda3\pkgs

我们直接将这两个目录压缩生成对应的压缩包：baseenv.zip 与 pkgs.zip

离线创建 conda 环境

在离线的机器上安装好 Anaconda 后，在 Anaconda Prompt (anaconda3) 执行以下命令将其设置为离线模式。

1

conda config --set offline true

然后创建需要的环境，以 new_env 作为环境名为例。

1

conda create -n new_env

环境恢复

将上述两个压缩包分别解压到离线机器的以下目录中：

压缩包解压到
baseenv.zipanaconda3\env\new_env\
pkgs.zipanaconda3\pkgs\

测试

经过上述步骤，环境就已经恢复成功了，可以通过命令行运行 python 并引入包的方式来进行测试。
